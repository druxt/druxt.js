# DruxtNuxtModule(moduleOptions)

Nuxt module function to install Druxt.

Kind: global function

Param Type Description
moduleOptions ModuleOptions DruxtJS module options.

Example (Nuxt configuration with module options)

module.exports = {
  modules: [
    ['druxt', { baseUrl: 'https://demo-api.druxtjs.org' }]
  ]
}
