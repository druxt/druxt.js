# NuxtJS

  1. Install Nuxt

  2. Install the Druxt module:

    npm i druxt
    1

  3. Add the module and configuration to nuxt.config.js:

    module.exports = {
  modules: [
    'druxt'
  ],
  druxt: {
    baseUrl: 'https://demo-api.druxtjs.org'
  }
}
